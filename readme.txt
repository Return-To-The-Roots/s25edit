About:
------
This is a re-engineered version of the Bluebytes (C) Siedler 2 Gold (C) Mapeditor.
Used librarys: SDL and a modified version of SGE.

Installation:
-------------
Put s2project.exe, SDL.dll, SGE.dll.
Don't create a subfolder. All files have to be in the same directory as your S2.EXE is.
